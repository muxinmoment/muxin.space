# 个人网站搭建指南

## 选型结论

- 内容站优先选 `Astro`
- 主页、分享页、项目页、关于页都用静态页面先搭起来
- 后面如果需要局部交互，再按需加 React 组件

## 当前结构

```text
src/
  content.config.ts
  content/notes/
  data/
    profile.ts
    projects.ts
    timeline.ts
    recommendations.ts
    principles.ts
  layouts/BaseLayout.astro
  pages/
    index.astro
    notes/
      index.astro
      [slug].astro
    projects.astro
    about.astro
    404.astro
  styles/global.css
```

## 本地开发

```bash
npm install
npm run dev
```

## 构建验证

```bash
npm run build
```

## Cloudflare Pages 配置

- Framework preset: `Astro`
- Build command: `npm run build`
- Build output directory: `dist`
- Node version: `20` 或 `22`

## 如何新增一篇分享

1. 在 `src/content/notes/` 新建一个 `.md` 文件
2. 写好 frontmatter:

```md
---
title: "文章标题"
description: "一句话摘要"
pubDate: 2026-05-31
tags:
  - astro
  - frontend
---
```

3. 写正文
4. 保存后执行 `npm run build` 验证
